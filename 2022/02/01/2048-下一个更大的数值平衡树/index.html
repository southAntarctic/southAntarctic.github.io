2048_下一个更大的数值平衡树

难度：中等

题目

如果整数 x 满足：对于每个数位 d ，这个数位 恰好 在 x 中出现 d 次。那么整数 x 就是一个 数值平衡数 。

给你一个整数 n ，请你返回 严格大于 n 的 最小数值平衡数 。

示例

示例一：

输入：n = 1
输出：22
解释：
22 是一个数值平衡数，因为：

  • 数字 2 出现 2 次
    这也是严格大于 1 的最小数值平衡数。

示例一：

输入：n = 1000
输出：1333
解释：
1333 是一个数值平衡数，因为：

  • 数字 1 出现 1 次。
  • 数字 3 出现 3 次。
    这也是严格大于 1000 的最小数值平衡数。
    注意，1022 不能作为本输入的答案，因为数字 0 的出现次数超过了 0 。

示例三：

输入：n = 3000
输出：3133
解释：
3133 是一个数值平衡数，因为：

  • 数字 1 出现 1 次。
  • 数字 3 出现 3 次。
    这也是严格大于 3000 的最小数值平衡数。

提示

  • $0 <= n <= 10^6$

解题

简单理解就是每个数字要在整数中出现它自身数值的个数

又因为这个数要大于 n

可以对数从 n+1 开始遍历，每次检查一下是否符合，否则数值 +1
